In an introductory database systems course, the assignment was to create a relational schema in 3rd Normal Form. Starting from scratch with no working design, the student was unsure how to represent the many-to-many relationship between 'Students' and 'Courses'. They asked an AI assistant to explain the concept of a join table. The AI described using a third table with foreign keys linking back to the student and course tables. Based on this initial explanation, the student proceeded to manually author the CREATE TABLE statements for the 'Students', 'Courses', and a new 'Enrollments' join table, directly applying the described technique.
